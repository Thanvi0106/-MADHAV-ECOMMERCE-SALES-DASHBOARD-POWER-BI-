# Madhav Ecommerce Sales Dashboard 📊

This project presents an interactive **Power BI dashboard** designed to analyze and visualize ecommerce sales data for the Madhav store. The dashboard provides insights into overall business performance using key metrics and visual analytics.The dashboard helps track sales performance, profit trends, product demand, and customer purchasing behavior. It transforms raw sales data into meaningful insights that support better business decision-making.

## Key Metrics

* **Total Sales Amount:** 438K
* **Total Profit:** 37K
* **Total Quantity Sold:** 5615
* **Average Order Value (AOV):** 121K

## Dashboard Features

* **Sales by State:** Identifies top performing regions such as Maharashtra and Madhya Pradesh.
* **Profit by Month:** Shows monthly profit trends and highlights high and low performance periods.
* **Quantity by Payment Mode:** Analyzes customer payment preferences including COD, UPI, Debit Card, Credit Card, and EMI.
* **Quantity by Category:** Compares demand across product categories like Clothing, Electronics, and Furniture.
* **Profit by Sub-Category:** Highlights profitable product groups such as Printers and Bookcases.

## Insights

* Clothing contributes the highest share of sales quantity.
* COD is the most commonly used payment method.
* Certain months show negative profits, indicating potential operational or pricing issues.
* Specific product sub-categories drive the majority of profit.

## Tools Used

* **Power BI**
* Data Cleaning and Transformation
* Data Visualization and Dashboard Design


This project demonstrates how data visualization can be used to monitor ecommerce performance, discover trends, and support data-driven business strategies.
